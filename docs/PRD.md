# MDViewer — Product Requirements Document

## 개요

MDViewer는 마크다운 문서를 아름답게 렌더링하는 읽기 전용 데스크톱 뷰어입니다.
편집 기능 없이 렌더링 품질에 집중합니다.

## 핵심 가치

**아름다운 렌더링** — 깔끔한 타이포그래피, 잘 정돈된 레이아웃, 세련된 색상.
마크다운 문서를 읽는 것이 즐거운 경험이 되어야 합니다.

## 대상 사용자

마크다운 문서를 자주 읽는 개발자 및 기술 문서 작성자.
macOS 사용자.

## 기능 요구사항

### F1. 마크다운 렌더링

지원하는 마크다운 요소:

| 요소 | 설명 |
|------|------|
| 제목 (H1~H6) | 모듈러 스케일 기반 크기 체계 |
| 단락 | 본문 텍스트, 적절한 행간 |
| 볼드/이탤릭/취소선 | 인라인 서식 |
| 순서 있는/없는 목록 | 중첩 지원 |
| 체크리스트 | 읽기 전용 체크박스 |
| 테이블 | GFM 테이블, 정렬 지원 |
| 코드 블록 | Syntax highlighting (GitHub 스타일) + 언어 표시 |
| 인라인 코드 | 배경색으로 구분 |
| 인용문 | 좌측 보더, 들여쓰기 |
| 수평선 | 구분선 |
| 링크 | 내부(.md) / 외부(URL) 구분 처리 |
| 이미지 | 로컬/원격 모두 지원, 반응형 |
| 수식 | KaTeX 기반, 인라인(`$...$`) + 블록(`$$...$$`) |
| Mermaid | 다이어그램 SVG 렌더링 |

### F2. 파일 열기

- `.md` 파일 연결 (File Association) — 더블클릭으로 MDViewer에서 열기
- Drag & Drop — 앱 윈도우에 파일 드래그
- File > Open (`Cmd+O`) — 파일 선택 다이얼로그
- 외부 앱에서 파일 열기 요청 수신

### F3. 탭

- 다중 탭 지원 — 여러 문서를 탭으로 전환
- 탭 바 — 파일명 표시, 닫기 버튼
- 탭 전환 — `Cmd+1~9`, `Cmd+Shift+[/]`
- 마지막 탭 닫기 시 Welcome 화면 표시

### F4. Quick Open (`Cmd+P`)

- 최근 열었던 파일 퍼지 검색
- 기본 정렬: 열었던 시간 역순 (최근 파일 상단)
- 검색어 입력 시 파일명/경로로 퍼지 매칭
- 이력 상한: 100개

### F5. 링크 동작

- `.md` 상대경로 링크 클릭 → 새 탭에서 열기
- 외부 URL 클릭 → 기본 브라우저에서 열기
- `#anchor` 링크 → 같은 문서 내 스크롤

### F6. 테마

- 라이트 / 다크 모드 — macOS 시스템 설정 자동 연동
- 테마 시스템 분리 — 폰트, 색상, 간격 등 독립적으로 튜닝 가능
- 테마는 TypeScript 객체로 정의, CSS Variables로 런타임 적용

### F7. 이미지 처리

- 로컬 상대 경로: 마크다운 파일 기준으로 절대 경로 변환
- 외부 URL: 그대로 로드
- 반응형: `max-width: 100%`로 컨테이너에 맞춤

## 비기능 요구사항

### 보안
- contextIsolation: true, nodeIntegration: false
- 모든 파일 I/O는 IPC 경유 (preload + contextBridge)
- CSP 헤더 설정

### 성능
- 일반적인 마크다운 문서 (1,000줄 이하) 즉시 렌더링
- 대용량 문서도 스크롤 버벅임 없이 동작

### 폰트
- 앱에 번들 (오프라인 지원)
- 본문: Inter (영문) + Pretendard (한글)
- 코드: JetBrains Mono
- woff2 포맷

## 범위 외 (Out of Scope)

- 마크다운 편집 기능
- 파일 저장/내보내기
- 사이드바/파일 트리
- 멀티 윈도우
- 플러그인 시스템
- 설정 UI (향후 확장 가능, 현재는 테마 파일 직접 수정)
