# /test-eval — 품질 평가

## 사용법
```
/test-eval    # 전체 품질 평가 실행
```

## 실행
@qa-evaluator 서브에이전트를 호출하여 10점 만점으로 평가합니다:

| 항목 | 배점 |
|------|------|
| 테스트 통과 | 2점 |
| 엣지 케이스 | 1점 |
| 린트 | 1점 |
| 최소 코드 | 2점 |
| 함수형 스타일 | 1점 |
| 방어 코드 | 1점 |
| Electron 보안 | 1점 |
| 회귀 안전 | 1점 |

7점 이상: PASS → PR 생성 가능
7점 미만: FAIL → 이슈 수정 필요
